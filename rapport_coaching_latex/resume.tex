\chapter*{Résumé}
\addcontentsline{toc}{chapter}{Résumé}

Ce rapport décrit la conception et le développement d'une plateforme de coaching sportif ("CoachingApp"). Afin d'améliorer la relation entre les coachs professionnels et leurs athlètes, l'application centralise la création et l'assignation de programmes d'entraînement interactifs, le suivi nutritionnel quotidien et la messagerie instantanée. 

Le système repose sur une architecture moderne à trois niveaux (3-Tiers), composée d'une application mobile cross-platform développée en Flutter pour les athlètes, d'un tableau de bord web d'administration conçu avec Angular, et d'une API robuste sous Node.js adossée à une base de données PostgreSQL. Une intégration d'intelligence artificielle est également proposée pour assister la génération de conseils.

\textbf{Mots-clés :} Coaching Sportif, Application Mobile, Flutter, Angular, Node.js, PostgreSQL, Scrum, Intelligence Artificielle.

\vspace{2cm}

\begin{center}
    \textbf{Abstract}
\end{center}

This report describes the design and development of a sports coaching platform ("CoachingApp"). To improve the relationship between professional coaches and their athletes, the application centralizes the creation and assignment of interactive training programs, daily nutritional tracking, and instant messaging.

The system relies on a modern three-tier architecture, consisting of a cross-platform mobile application developed in Flutter for athletes, an administrative web dashboard designed with Angular, and a robust Node.js API backed by a PostgreSQL database. An artificial intelligence integration is also provided to assist in generating advice.

\textbf{Keywords:} Sports Coaching, Mobile Application, Flutter, Angular, Node.js, PostgreSQL, Scrum, Artificial Intelligence.
